    5.z-index的问题
      必须有定位属性，才能设置z-index 值越大，优先显示，层级越高（离你越近）
    6.布局技巧
    A）
  1）页面中的布局，都是由盒子组成的
  2）写布局的时候，先布局全局，在调整局部
  3）布局的时候，先用背景颜色，把位置站好
    B）
  1）左浮动
  2）右浮动
  3）清除浮动（最好的方式，通过在父级设置一个类（伪元素））
    说明：（什么情况用浮动）
      传统布局，块级元素，只要出现两栏以上布局，就要用浮动。
    .clearfix::after {
    content:'';
    display:inline-block;
    height: 0;
    clear: both;
  }v
    C）

  1）子绝父相
      子绝对定位    父相对定位
      注意￥￥￥：如果子元素是绝对定位，那么，他会相对最近的一个具有定位属性的父元素定位，
                  如果父元素都没有定位属性，那么就是相对浏览器定位。
  2) 固定定位：相对于浏览器窗口定位，不会随窗口滚动
  3）相对定位：相对于自己原来的位置定位
  postion:absolute;
  z-index:10;

 基本选择器
          id
          类

          通配符
          子代    ul>li  仅仅是ul下面的儿子这一带的li        
          后代     ul li  选中的是所有后代的li   选择的范围阔大

 :first-child()   // 第一个儿子
 :last-child()    //最后一个儿子
 :nth-child()     //第n个儿子
 :first-of-type   只认相同的儿子 列入ul li:first-of-type  只选中ul元素的第一个li儿子
